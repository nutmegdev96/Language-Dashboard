## 🎯 About The Project

**Linguasphere** is a cutting-edge, cyberpunk-styled dashboard that provides real-time intelligence about the world's most spoken languages. Built with a futuristic UI and powered by a Python FastAPI backend, it offers live data updates, historical trends, and even ML-powered predictions for language growth.

### ✨ Why Linguasphere?
- **Real-time Data** - Live updates via WebSocket connection
- **Futuristic UI** - Cyberpunk aesthetic with glitch effects and neon gradients
- **ML Predictions** - AI-powered language growth forecasts until 2050
- **Interactive Maps** - Visual representation of language distribution
- **Full Stack** - Beautiful frontend + Powerful Python backend

---

## 🚀 Features

### 📊 **Dashboard Components**
- **Live Statistics Cards** - Real-time counters with animations
- **Top 10 Languages Bar Chart** - Most spoken languages worldwide
- **Continent Distribution** - Doughnut chart showing language spread
- **Growth Trends** - Line chart tracking language evolution (2020-2024)
- **2030 Projections** - Radar chart with future predictions
- **World Language Map** - Interactive bubble map by continent

### ⚡ **Backend API (Python/FastAPI)**
- **WebSocket Connection** - Live data every 5 seconds
- **RESTful Endpoints** - Access all data via API
- **ML Predictions** - Language growth forecasts until 2050
- **Historical Data** - 10-year language trends
- **Broadcast System** - Push notifications to all clients

## 📦 Installation

### **Prerequisites**
- Python 3.9+
- pip
- Git

### **Clone & Setup**

```bash
# Clone the repository
git clone https://github.com/nutmegdev96/languages-dashboard.git
cd languages-dashboard

# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the server
python main.py
