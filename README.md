# 🌍 World Languages Dashboard

Dashboard interattiva che mostra le lingue più parlate nel mondo con grafici e statistiche.

## 🚀 Demo Live
[Visualizza il sito](https://nutmegdev96.github.io/languages-dashboard/)

## 📊 Funzionalità
- Top 10 lingue più parlate (bar chart)
- Distribuzione per continente (doughnut chart)
- Trend di crescita delle principali lingue (line chart)
- Mappa interattiva delle lingue per regione
- Statistiche in tempo reale con animazioni

## 🛠️ Tecnologie utilizzate
- HTML5
- CSS3
- JavaScript
- Chart.js
